# Three.js 僵尸射击游戏

基于 Three.js 的第三人称 3D 僵尸射击游戏，纯前端实现，无需后端服务。

## 快速开始

```bash
npm install
node server.js
```

浏览器打开 `http://localhost:3001` 即可游玩。

也支持手机访问（同一局域网内用电脑 IP 地址）。

## 关卡系统

游戏采用递增式关卡设计，第 N 关的难度参数如下：

| 参数 | 公式 | 第1关 | 第2关 | 第3关 | ... | 第N关 |
|------|------|-------|-------|-------|-----|-------|
| Boss 僵尸数量 | N | 1 | 2 | 3 | ... | N |
| 小僵尸总数 | 40 + N x 30 | 70 | 100 | 130 | ... | 40 + N x 30 |

**Boss 刷新规则：** Boss 逐个出现，不会一次全部刷出。场上已有 Boss 数量达到上限后不再刷新新的 Boss。

**过关条件：** 击杀当前关卡所有 Boss 和所有小僵尸（达到设定的击杀目标数即可过关，不需要等场上清空）。

**过关动画：** 角色从高处落下，生命恢复 25 点（不超过上限），进入下一关。

## Boss 僵尸

Boss 僵尸体型巨大（普通僵尸的 3~4 倍），具有以下特征：

- 移动缓慢（速度为普通僵尸的 40%）
- 大量生命值
- 红色发光眼睛
- **半血狂暴：** 血量降至 50% 以下时进入狂暴状态
  - 攻击力翻倍（两下即可击杀主角）
  - 眼睛变大发光
- 击杀后掉落 3 个道具（2 个多弹道 + 1 个攻速提升）

## 积分系统

| 目标 | 积分 |
|------|------|
| 击杀小僵尸 | +100 |
| 击杀 Boss 僵尸 | +500 |

积分跨关卡持续累积。死亡时显示总积分和表现评语。

## 道具系统

Boss 僵尸被击杀后掉落道具，角色靠近自动拾取：

| 道具 | 效果 |
|------|------|
| 多弹道（橙色） | 弹道数 +2，最高 11 发 |
| 攻速提升（绿色） | 射击间隔缩短 30% |

**过关后道具减半：** 每过一关，多弹道数量减半（至少保留 1），攻速倍率恢复 50%。这意味着需要持续拾取道具来维持火力优势。

## 战斗机制

### 爆头系统

击中小僵尸头部可一击秒杀，触发爆头特效：

- 屏幕震动
- 红色粒子爆发
- HEADSHOT! 文字提示
- 专用音效

### 受伤加速

僵尸血量越低移动越快：

- 满血时：正常速度
- 半血时：速度 x 1.25
- 濒死时：速度 x 1.5

配合关卡进度加速，后期的濒死僵尸移动非常快。

### 关卡进度加速

每关新刷出的僵尸自带速度加成，取决于当前关卡进度：

- 关卡开头：正常速度（x1.0）
- 关卡末尾：加速 80%（x1.8）

### 玩家死亡

当生命值归零时，玩家不会立即死亡，而是进入死亡动画：

1. 角色向后仰倒（1.5 秒）
2. 原地爆炸，产生 AOE 伤害（半径 10）
3. 范围内所有僵尸被秒杀

如果爆炸恰好清完了场上所有剩余僵尸，显示"LEVEL CLEAR"（本关胜利），否则显示"GAME OVER"。

## 程序结构

- `server.js` -- Express 静态文件服务器
- `package.json` -- 依赖声明
- `public/index.html` -- 游戏主文件（HTML + CSS + JS 全部内联）
- `three.min.js` -- Three.js r160（缓存副本，用于生成单文件版本）

## 技术栈

- **渲染引擎：** Three.js r160
- **音效：** Web Audio API（程序化生成，无音频文件）
- **后端：** Express.js（仅用于静态文件托管）
- **部署：** 可打包为单个 HTML 文件（约 758KB），无需任何依赖即可运行

## 单文件版本

项目支持生成包含 Three.js 内联的单文件版本，双击即可在浏览器中打开，无需联网、无需服务器。

## 移动端适配

自动检测触屏设备，PC 端使用键盘 + 鼠标，移动端显示虚拟摇杆和按钮。两种操作方式互不干扰。
